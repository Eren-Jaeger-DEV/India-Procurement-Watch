HOW TO USE:
  1. Copy your SQLite .db file(s) into THIS folder.
  2. Go to the dashboard at http://localhost:5000
  3. Click "Analyse Data" — that's it.

SUPPORTED FILES:
  • aoc_tenders.db   — Awarded contracts database (required)
  • tenders_vps.db   — Published tenders database (optional)

The tool will auto-detect any .db files placed here
and run the full analysis pipeline automatically.
